# GoCoin

GoCoin es una aplicación de finanzas personales diseñada para ayudarte a gestionar tus ingresos y gastos de manera eficiente y visual.

## Características

- Registro de transacciones (Ingresos y Gastos)
- Visualización de saldo mensual
- Filtros por fecha y categoría
- Interfaz intuitiva y moderna
- Soporte para múltiples divisas (COP por defecto)

## Tecnologías

- **Frontend:** React + Vite
- **Estilos:** Tailwind CSS / Vanilla CSS
- **Base de Datos:** Supabase
- **Despliegue:** Vercel

## Configuración Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/juliocuva/gocoin.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
